01 · About

The Operator

Located in the garden city of India, Bangalore. Licensed since 1993, I started operations in Thrissur, Kerala with a simple homebrew QRP rig following VU2VWN OM Vasanth (SK)'s circuit — the most popular design of those days.

Later graduated to SSB with the help of an old exciter originally made by VU2UCY Om Bhasi (SK). Went through different iterations of 807 tube QRO and some military surplus gear (Redifone GR 345) during the 1990s. Shifted to Bangalore in early 2003.

I travel extensively for work and mostly operate remote during free time. The shack runs a fully automated setup including remote antenna switching and lightning protection.

Utilities & Tools
FT8 Cluster Aggregator
Standalone macOS application that aggregates spots from multiple FT8 sources and DX cluster nodes, then re-serves them through a built-in telnet server that mimics a standard DX cluster. User login is supported via RumLog and FlexRadio. Spots are filtered against the current worked DXCC chart pulled from Club Log and displayed in a colour-coded interface. Based on original work by Vinod VU3ESV, modernised and extended.

SPE Expert Amplifier Control & Display
Web-based control and monitoring interface for the SPE Expert 1.5 KFA linear amplifier. A WebSocket server runs on a Raspberry Pi connected to the amplifier via USB, serving real-time data — power, SWR, temperature, band and operating state — to any browser on the network. Based on original work by OH2GEK, modernised and adapted for the Expert 1.5 KFA.

VU DXCC Credits Tracker
Live, sortable table of ARRL DXCC standings for every Indian (VU) callsign across all bands from 160 m to 6 m, plus Mixed / Phone / CW / Digital / Satellite / Challenge / Honor Roll. A GitHub Actions workflow runs monthly, downloads the 17 ARRL DXCC Standings PDFs, extracts every VU entry and republishes both a printable PDF and JSON data file consumed by the page.
